## 何谓学习

修学储能：

* 修学：即研修学问，积累知识。属理论层，是才能的基础。
* 储能：即储备才能，培养能力。属应用层，是知识的实践。

## 为何学习

* 使自己具备解决问题的能力。

## 如何学习

1. 学前-备
    1. 虚怀若谷，倒掉杯里的水才能装更多的水。 empty your mind.
    2. 博渊并重
    3. 先博后渊，量变引发质变。
2. 学中-进
    1. 用心专一
    2. 持之以恒
3. 学后-习
    1. 三省吾身
